Introduction

HTML forms are used to collect user input. They contain various form elements such as text fields, checkboxes, radio buttons, and submit buttons. The collected data can be sent to a server for processing.

Form Elements

  • Input

    The <input> element is one of the most commonly used form elements. It can be used for various types of input fields, such as text, password, email, and more, by setting the type attribute.

    <input type="text" placeholder="Enter your name">

  • Select

    The <select> element creates a drop-down list. It contains multiple <option> elements, each representing an item in the list. Users can select one or multiple options from the list. For example:

    <select>
     <option value="volvo">Volvo</option>
     <option value="saab">Saab</option>
     <option value="mercedes">Mercedes</option>
     <option value="audi">Audi</option>
    </select>

  • Textarea

    The <textarea> element provides a multi-line text input field. It is commonly used for collecting longer pieces of text, such as comments or descriptions.

    <textarea rows="4" cols="50" placeholder="Enter your message"></textarea>

  • Button

    The <button> element defines a clickable button. It can be used to submit forms, reset forms, or trigger other actions. Buttons can contain text, images, or other content.For example:

    <button type="submit">Submit</button>
Form Attributes

  • Action

    The action attribute specifies the URL where the form data will be sent for processing. It is a required attribute for form submission.

    <form action="/submit-form.php" method="post">

  • Method

    The method attribute specifies how the form data should be sent to the server. Common values are GET and POST. GET appends form data to the URL, while POST sends it as a part of the request body.For example:

    <form method="post">

  • Target

    The target attribute specifies where to display the response after submitting the form. Common values are _self (default), _blank, _parent, and _top.

    <form target="_blank">

  • Enctype

    The enctype attribute specifies the encoding type of the form data when submitting it to the server. Common values are application/x-www-form-urlencoded, multipart/form-data, and text/plain. For example:

    <form enctype="multipart/form-data">
Input Types and Attributes

  • Text

    The type="text" input field is used for single-line text input.

    <textarea rows="4" cols="50" placeholder="Enter your message"></textarea>

  • Password

    The type="password" input field hides the entered characters, making it suitable for sensitive information such as passwords.

    <input type="password" name="password">

  • Email

    The type="email" input field is used for email addresses. It ensures that the entered value conforms to the email format.

    <input type="email" name="email">

  • Number

    The type="number" input field is used for numeric input. It can have additional attributes like min, max, and step.

    <input type="number" name="quantity" min="1" max="100" step="1">

  • Date

    The type="date" input field provides a date picker to select a date.

    <input type="date" name="birthdate">

  • Radio

    The type="radio" input field is used to select one option from a group. Multiple radio buttons with the same name attribute form a group.

    <input type="radio" name="gender" value="male"> Male
    <input type="radio" name="gender" value="female"> Female

  • Checkbox

    The type="checkbox" input field allows users to select multiple options independently.For example:

    <input type="checkbox" name="vehicle1" value="Bike"> I have a bike<br>
    <input type="checkbox" name="vehicle2" value="Car"> I have a car<br>
Form Validation

  • Required

    The required attribute specifies that an input field must be filled out before submitting the form.

    <input type="text" name="username" required>

  • Pattern

    The pattern attribute specifies a regular expression that the input field's value must match for the form to be submitted.

    <input type="text" name="username" pattern="[A-Za-z]{3,}" title="Three or more letters">

  • Min and Max

    The min and max attributes specify the minimum and maximum values for numeric and date input fields.For example:

    <input type="number" name="quantity" min="1" max="10">

  • Step

    The step attribute specifies the legal number intervals for numeric input fields. It is used to control the granularity of the value. For example, if step="0.01", the input field will allow values with two decimal places.

    <input type="number" name="price" step="0.01">
